3.教务管理员用户操作

教务管理员部分主要有个人中心、测试管理、论坛管理、资源管理、教务管理、机构管理、用户管理、教务统计等八个模块，主要用于教务管理员用户对教师用户以及学生用户的教学管理，资源管理，用户管理，交流管理，测试管理，机构管理以及通知消息的接收和发送等。

3.1 访问教务管理员个人中心

教务管理员个人中心由两个部分组成：个人中心，消息中心。个人中心主要是显示教务管理员用户需求量高的两个模块（最新通知、近期资源更新），而消息中心主要起到显示教务管理员用户所有的消息通知以及教务管理员发送通知等作用。

3.1.1 查看教务管理员最新通知、近期资源更新

教务管理员登录系统后，默认进入教务管理员个人中心。个人中心主要包括：最新通知、近期资源更新等。教务管理员个人中心界面如图3-1所示：

图3-1 个人中心－个人中心

3.1.2 查看教务管理员通知消息、管理通知消息

在个人中心一级栏目中选择二级栏目消息中心即可进入消息中心。消息中心主要包括：教务员管理员所有的通知消息。与教师、学生用户不同的是，教务管理员有发送通知消息的权限。消息具体信息和学生、教师部分相似，在此就不一一列举了，教务管理员消息中心界面如图3-2所示：

3.2 进行测试管理

测试管理由四个二级栏目组成：课程测试管理，题库管理，试卷管理，成绩管理。题库管理是放在第一位的，首先教师需要添加题库才能构成试卷；其次是试卷管理，试卷管理分自动组卷和手动组卷，这样合成了一份份试卷；接下来的课程测试管理，这张试卷什么时候考试，怎么考，都有课程测试管理来决定。最后成绩管理很明显是对学生的成绩进行管理。整个测试管理模块与教师用户部分的测试管理相同，这里引用教师测试管理部分演示。

3.4 管理课程资源

        教务管理员的资源管理是针对所有上传的给学生授课的所需资源进行的管理，包括对视频的管理、文档的管理、链接的管理。与教师部分不同的是，教务管理员不必对课程的章节内容进行管理。教务管理员对资源的管理方便了对于区域教学资源的管理控制，便于区域资源共享。由于视频管理、文档管理、链接管理这三个部分的具体操作与教师部分完全相同，这里以教师管理课程资源部分来举例。

图 教师-14 资源管理－视频管理－添加视频资源

        点击视频资源主界面标题即可编辑视频资源，编辑步骤：修改需要修改的部分，如需更改视频资源上传资源即可；若不需更改视频资源，则不用上传资源（注：视频格式只能是flv），上传视频完成后会返回“上传成功”，此时才可以进行添加视频资源信息的确认，具体界面由图 教师-15所示：
