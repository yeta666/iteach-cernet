3.系统管理员用户操作

系统管理员部分主要有个人中心、学习中心、课程讨论、我的测试、我的课堂、测试管理、论坛管理、资源管理、教务管理、机构管理、用户管理、系统管理、教务统计等十三个模块，主要用于系统管理员用户对教务管理员、教师用户以及学生用户还有系统整体信息、功能的管理，有资源管理，用户管理，交流管理，测试管理，机构管理以及通知消息的接收和发送、编辑等。

3.1 访问系统管理员个人中心

系统管理员个人中心由两个部分组成：个人中心，消息中心。个人中心主要是显示系统管理员用户需求量高的两个模块（最新通知、服务器状态信息），而消息中心主要起到显示系统管理员用户所有的消息通知以及系统管理员发送及编辑通知等作用。

3.1.1 查看系统管理员最新通知、服务器状态信息

系统管理员登录系统后，默认进入系统管理员个人中心。个人中心主要包括：最新通知、服务器状态信息等。系统管理员个人中心界面如图7-1所示：

3.6 进行测试管理

测试管理由四个二级栏目组成：课程测试管理，题库管理，试卷管理，成绩管理。题库管理是放在第一位的，首先教师需要添加题库才能构成试卷；其次是试卷管理，试卷管理分自动组卷和手动组卷，这样合成了一份份试卷；接下来的课程测试管理，这张试卷什么时候考试，怎么考，都有课程测试管理来决定。最后成绩管理是对学生的成绩进行管理。由于此部分功能操作与教师部分相同，这里就以教师部分为例。测试管理主界面如下图所示：

3.8 管理课程资源

        资源管理是针对所有上传的给学生授课的所需资源进行的管理，包括对视频的管理、文档的管理、链接的管理。最后再在章节管理中汇总，让资源管理与实际章节操作相链接。这里与教师的资源管理不同在于多了一个服务器管理，便于系统管理员控制服务器信息。其余的四项与学生部分完全相同，这里就以学生部分举例。
